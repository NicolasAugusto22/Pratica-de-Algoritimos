Conversor de bases numéricas desenvolvido com HTML, CSS e JavaScript.  
Permite transformar números decimais em binário, octal e hexadecimal de forma simples e didática.

Digite um número decimal e veja instantaneamente sua conversão para outras bases numéricas.

Este projeto trabalha conceitos fundamentais da computação:
- Sistema decimal (base 10)
- Sistema binário (base 2)
- Sistema octal (base 8)
- Sistema hexadecimal (base 16)
- Conversão entre bases numéricas

Tecnologias utilizadas:
- HTML
- CSS
- JavaScript
